NBA Canlı Oyuncu Takip 🏀

Bu uygulama, canlı NBA maçlarını takip ederek seçtiğiniz oyuncuların istatistiklerini (Sayı, Asist, Ribaund vb.) anlık olarak masaüstünüzde izlemenizi sağlayan profesyonel bir asistandır. Canlı yayın ile API arasındaki gecikmeyi (delay) hesaplayarak yayınınıza tam senkronize şekilde çalışır.

🚀 Özellikler

Anlık Veri: NBA resmi API'si üzerinden saniyesi saniyesine istatistik takibi.

Oto Gecikme Bulucu: Yayınınızın geriden gelmesi durumunda, ekrandaki skora göre otomatik "delay" hesaplama.

Yatay Mini Mod (Floating Cards): Oyun oynarken veya yayın izlerken ekranın köşesinde her zaman üstte duran şeffaf ve yatay bilgi kartları.

Akıllı Hafıza: Uygulamayı kapattığınızda takip ettiğiniz oyuncular ve ekrandaki kart pozisyonlarınız otomatik kaydedilir.

Görsel Bildirimler: Oyuncu sayı attığında veya barem aşıldığında dinamik renk değişimleri (Yeşil/Kırmızı).

📥 İndir ve Kullan

Uygulamayı hiçbir kod veya kurulumla uğraşmadan direkt kullanmak için:

Sağ taraftaki Releases sekmesine tıklayın.

En son sürümün altındaki BetApp.exe dosyasını indirin.

Çift tıklayıp anında kullanmaya başlayın!

💻 Geliştiriciler İçin Kurulum

Projeyi kaynak kodundan çalıştırmak isterseniz:

Python yüklü olduğundan emin olun.

Depoyu bilgisayarınıza indirin ve klasörde terminal (CMD) açın.

Sanal ortam oluşturup kütüphaneleri yükleyin:

python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt


Uygulamayı başlatın: python BetApp.py
